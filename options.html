这个应用是基于jinntrance的扇贝查词应用做的。

改进的动力：第一代码（我是专职前端，对js相对熟悉点，但是是野生的，技术跟jinntrance没得比……）；第二扇贝API升级了，并把之前的API标记为废弃了；第三我想做个扩展玩玩。

本项目的 GitHub地址。感觉有帮助的话给个赞，嗯嗯。

反馈

反馈有两种方式：邮件和GitHub issue。建议使用GitHub。

1, 本人邮箱

如果电脑上有邮箱应用并添加了邮箱账号，可以直接点击，主题为也已经在链接里写好了。如果没有，主题一定要写成“Chrome-shanbay-extension-issue”。

⚠️邮件内容一定要包括：操作系统的名称、版本，chrome的名称（有正式版，开发版，金丝雀版等等）、版本号。尽量描述清楚问题出现的原因，最好有配图，GIF最好^ _ ^，因为我不可能说：“算了，我自己看看你的浏览器吧”。

2, GitHub issue

issue的内容也尽量按照上面的来，我就不再说一遍了。这样我方便找问题解决问题，你也能尽快得到一个更好的插件。

最后，祝你们插件用得愉快，英语进步快，不要像我，老大背单词(╯°□°）╯︵ ┻━┻。
